❤️ C O L O R I N E ❤️

Le but du jeu est de mémoriser l'emplacement des cases de couleurs. Une fois celles-ci mémorisées, vous n'avez plus qu'à appuyer sur le bouton Mémorisé.

Par la suite, ces cases vont disparaitre et vous devez les retrouver en cliquant dessus.

  • Si vous vous trompez, vous devez recommencer.
  • Si vous réussissez, vous passez à l'étape suivante.
